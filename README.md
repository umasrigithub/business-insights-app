# Vibe Coded Experience – Unified Analytics View

This is a compact Streamlit app designed for job applications that ask for a "vibe coded experience."

## What it shows
- KPI framing for leadership
- A unified view across revenue, channels, retention, and product mix
- Business insight storytelling, not just charts
- A concise explanation of working style and decision-making approach

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Community Cloud
1. Create a new GitHub repository.
2. Upload `app.py`, `requirements.txt`, and `README.md`.
3. Push to GitHub.
4. Go to Streamlit Community Cloud.
5. Select your repo and deploy `app.py`.
6. Copy the public app URL and submit that link.

## Suggested application note
I approached this experience as a compact analytics narrative rather than a generic dashboard. The goal was to show how I think: start with business questions, organize fragmented data into a clear KPI structure, diagnose performance drivers, and communicate insights in a way leadership can act on quickly.
